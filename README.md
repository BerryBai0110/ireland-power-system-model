# Ireland Power System Model

## Overview

This project is an interactive physical demonstration model of the Irish electrical power transmission system.

The system combines:

- 3D printed terrain of Ireland
- Wind turbine generation models
- Power transmission visualization
- LED load indicators
- Arduino and Raspberry Pi integration
- Renewable energy flow demonstrations

The model is designed for engineering education, renewable energy visualization, and interactive outreach demonstrations.

---

## Project Goals

The project aims to demonstrate how electrical power is:

1. Generated from renewable energy sources
2. Transmitted through the Irish national grid
3. Distributed to major population centres
4. Visualized using interactive hardware systems
